Do's & Don'ts

✓ Do
  • Use the correct portal color variant for each app
  • Maintain equal padding around the logo (minimum 8px clear space)
  • Use logo-master.svg for printed/PDF/email contexts
  • Import only from shared/logo/ — use the Vite alias
  • Use <img> or inline SVG — both are fine
  • Import banners (banner-*.svg) for sidebar desktop headers
  • Maintain aspect ratio at all times
✗ Don't
  • Don't copy SVG files into individual portal src/assets/ folders
  • Don't stretch or squash the logo (set both width and height equally)
  • Don't recolor or modify the SVG gradients
  • Don't place the backoffice (blue) logo in the client app
  • Don't render below 16px (favicon minimum)
  • Don't use the letter "O" div or Lucide icons as logo substitutes
  • Don't place the logo on clashing mid-tone colored backgrounds

React Import Examples

Copy-paste snippets for each portal

1 · Import the logo URL

// Works in all portals via the @shared-logo Vite alias
import orderiumLogo from '@shared-logo/logo-backoffice.svg';   // backoffice
import orderiumLogo from '@shared-logo/logo-client.svg';       // client portal
import orderiumLogo from '@shared-logo/logo-delivery.svg';     // delivery portal
import orderiumBanner from '@shared-logo/banner-backoffice.svg'; // sidebar banner

2 · Sidebar header (desktop expanded)

// Shows full banner with wordmark on expanded sidebar
<img
  src={orderiumBanner}
  alt="Morocom"
  style={{ height: '2rem', width: 'auto' }}
/>

3 · Sidebar header (collapsed / icon-only)

<img
  src={orderiumLogo}
  alt="Morocom"
  style={{ width: '2.25rem', height: '2.25rem', flexShrink: 0 }}
/>

4 · Login page hero logo

<img
  src={orderiumLogo}
  alt="Morocom"
  style={{
    width: '90px',
    height: '90px',
    marginBottom: '1.25rem',
    position: 'relative',
    zIndex: 1,
    filter: 'drop-shadow(0 8px 30px rgba(0,0,0,0.22))',
  }}
/>

5 · Mobile header logo + wordmark 

// Logo-only on mobile (sm:hidden), banner on desktop (hidden sm:flex)
<Link to="/">
  </* Mobile: logo icon only */>
  <img src={orderiumLogo} alt="Morocom"
       style={{ width: '2.25rem', height: '2.25rem' }} />

  </* Desktop: full banner */>
  <img src={orderiumBanner} alt="Morocom"
       className="hidden sm:block"
       style={{ height: '2rem', width: 'auto' }} />
</Link>

6 · TypeScript type declaration (if needed)

// vite-env.d.ts or a custom *.d.ts file
declare module '*.svg' {
  const src: string
  export default src
}
