# Morocom Delivery Portal

A delivery person portal for managing assigned orders with real-time updates and navigation integration.

## Features

- 🔐 **Authentication**: Secure login for delivery personnel
- 📦 **Order Management**: View and manage assigned deliveries
- 🗺️ **Navigation Integration**: One-click navigation via Google Maps or Waze
- ⏱️ **Real-time Updates**: Auto-refresh every 30 seconds
- 📱 **Mobile-First Design**: Optimized for field use
- 🎨 **Moroccan-Inspired UI**: Consistent design with the main client portal

## Design System

The delivery portal shares the same Moroccan-inspired design system as the client portal:

- **Color Palette**: Terracotta primary, warm sand secondary, gold accents
- **Typography**: Outfit font for LTR, Noto Sans Arabic for RTL
- **Components**: shadcn/ui components with custom theming
- **Animations**: Smooth transitions and fade-in effects

## Tech Stack

- **Framework**: React 18.2 + TypeScript 5.2.2
- **Build Tool**: Vite 5.0.8
- **Routing**: React Router DOM 6.20
- **State Management**: TanStack React Query 5.13.4
- **Styling**: Tailwind CSS 3.3.6 with custom Moroccan theme
- **UI Components**: shadcn/ui (Radix UI primitives)
- **Icons**: Lucide React + React Icons

## Getting Started

### Installation

```bash
cd delivery-portal
bun install
```

### Development

```bash
bun dev
```

The portal will run on [http://localhost:3003](http://localhost:3003)

### Build

```bash
bun build
```

## API Integration

The portal integrates with the shared server API:

- **Base URL**: `http://localhost:5000`
- **Login**: `POST /api/delivery/login`
- **Get Orders**: `GET /api/delivery/orders`
- **Update Status**: `PUT /api/delivery/orders/:id/status`

## Order Status Flow

1. **Assigned** → Order assigned by admin
2. **In Delivery** → Delivery person starts delivery
3. **Delivered** → Order successfully delivered

## Features by Page

### Login Page
- Phone number + password authentication
- Error handling
- Responsive design
- Moroccan-themed gradient background

### Orders Page
- **Header**: Welcome message, sign out button
- **Filter Tabs**: All, Assigned, In Delivery, Delivered
- **Order Cards**: 
  - Customer contact info
  - Delivery address
  - Google Maps & Waze navigation buttons
  - Order items and total
  - Status update buttons
- **Auto-refresh**: Updates every 30 seconds
- **Empty states**: Friendly messages when no orders

## Color Theme

The delivery portal uses a terracotta-based Moroccan theme:

```css
--primary: 15 65% 50%;        /* Terracotta */
--secondary: 35 35% 88%;      /* Warm sand */
--accent: 42 85% 55%;         /* Gold */
--background: 35 30% 97%;     /* Light beige */
--foreground: 25 40% 15%;     /* Dark brown */
```

## Navigation Integration

Each order card includes direct links to:
- **Google Maps**: Opens address in Google Maps app
- **Waze**: Opens address in Waze app

URLs are pre-generated by the server and include coordinates for accurate navigation.

## Session Management

- Authentication tokens stored in localStorage
- Persistent login across page refreshes
- Automatic logout functionality
- Protected routes redirect to login if not authenticated

- Mobile-first approach
