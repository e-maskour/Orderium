## Getting Started

### Installation

```bash
cd delivery-portal
bun install
```

### Development

```bash
bun dev
```

The portal will run on [http://localhost:3003](http://localhost:3003)

### Build

```bash
bun build
```

## API Integration

The portal integrates with the shared server API:

- **Base URL**: `http://localhost:5000`
- **Login**: `POST /api/delivery/login`
- **Get Orders**: `GET /api/delivery/orders`
- **Update Status**: `PUT /api/delivery/orders/:id/status`

## Order Status Flow

1. **Assigned** â†’ Order assigned by admin
2. **In Delivery** â†’ Delivery person starts delivery
3. **Delivered** â†’ Order successfully delivered

## Features by Page

### Login Page
- Phone number + password authentication
- Error handling
- Responsive design
- Moroccan-themed gradient background

### Orders Page
- **Header**: Welcome message, sign out button
- **Filter Tabs**: All, Assigned, In Delivery, Delivered
- **Order Cards**: 
  - Customer contact info
  - Delivery address
  - Google Maps & Waze navigation buttons
  - Order items and total
  - Status update buttons
- **Auto-refresh**: Updates every 30 seconds
- **Empty states**: Friendly messages when no orders

## Color Theme

The delivery portal uses a terracotta-based Moroccan theme:

```css
--primary: 15 65% 50%;        /* Terracotta */
--secondary: 35 35% 88%;      /* Warm sand */
--accent: 42 85% 55%;         /* Gold */
--background: 35 30% 97%;     /* Light beige */
--foreground: 25 40% 15%;     /* Dark brown */
```

## Navigation Integration

Each order card includes direct links to:
- **Google Maps**: Opens address in Google Maps app
- **Waze**: Opens address in Waze app

URLs are pre-generated by the server and include coordinates for accurate navigation.

## Session Management

- Authentication tokens stored in localStorage
- Persistent login across page refreshes
- Automatic logout functionality
- Protected routes redirect to login if not authenticated

## Responsive Design

- Mobile-first approach
- Optimized for field use on smartphones
- Touch-friendly buttons and cards
- Smooth scrolling and transitions
